## How to Run

Also no git ignore created so .env file is there.
```bash
npm install
npm run start:all
```

Open http://localhost:3000

App at:
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000/api/activities

## I built

- React frontend with activity form, list, and filter components
- Express.js backend with MongoDB Atlas for persistent storage
- CRUD operations: add, view, filter, delete activities
- Four categories: Academic, Technical, Cultural, Sports
- Real-time filtering and deletion
- Date validation (no future dates allowed)
- Responsive design for desktop and mobile
- Automatic port cleanup on startup
